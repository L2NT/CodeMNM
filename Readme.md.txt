# Spotify Clone 🎧

A simple music streaming web application inspired by Spotify.

## 🎯 Features

- 🎵 Play/pause songs
- 🔍 Browse and search music
- ❤️ Like/favorite tracks
- 📃 Playlist management
- 🖥️ Responsive UI (mobile + desktop)

## 🛠️ Tech Stack

- Frontend: ReactJS, bootstrap
- Backend: Django
- Database: SQL
- Authentication: Authenticated Django
